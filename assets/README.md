# Assets Folder

Place your AI service logo PNG images here with these exact filenames:

- `chatgpt.png` - ChatGPT logo
- `claude.png` - Claude logo  
- `gemini.png` - Gemini logo
- `grok.png` - Grok logo
- `github.png` - GitHub logo
- `linkedin.png` - LinkedIn logo

**Recommended specs:**
- Format: PNG with transparent background
- Size: 128x128px to 256x256px (square)
- Quality: High resolution for crisp display

**Where to get logos:**
1. Download official brand assets from each company's press/brand kit
2. Use high-quality icon packs (ensure licensing permits your use)
3. Extract favicons from the actual websites (lower quality)

If an image is missing, the white circle will still display empty but clickable.
